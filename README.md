# TerraCognita Toolset

For the successful usage of the Terra Cognita Toolset Mental Mapping Tool you need the followings:
- Google Developer profile 
- API Key for Google Maps
- Custom map style within Google Maps
- database server

Don't forget to include your custom details in the JS and HTML files. 

The SQL file should be run on the database server to create the "places" table.

The other four files should be uploaded to your webserver.
